# Learning journal

One entry per week, written while it's fresh (Sunday slot). Copy the template below to `week-NN.md`. The dated commits are the point.

## Template

```markdown
# Week NN — <title>

**Built:** what shipped this week (link to repo/commit)
**Learned:** 2–3 things that actually surprised me
**Broke:** what went wrong and how I fixed it
**Next:** one sentence
**Posted:** <LinkedIn post link>
```
